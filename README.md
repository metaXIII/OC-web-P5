# Sudoku Solver

Sudoku solver written in PHP

### Docker-compose

```
docker-compose up -d
```

## Docker

### Build my application

```
docker build -t p5 .
```

### Build and run the application

```
docker run -d -p 80:80 -v ${pwd}:/var/www/html p5
```
